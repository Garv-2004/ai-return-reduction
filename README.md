# AI-Based Return Reduction for E-commerce

## Problem Statement

This project predicts the likelihood of product returns before purchase using machine learning.
It helps reduce return rates by analyzing user behavior, product attributes, and historical return data.

## Features

* Predicts return probability
* Suggests recommendation to reduce returns
* API support using FastAPI
* Frontend using Streamlit

## Tech Stack

* Python
* Pandas
* Scikit-learn
* FastAPI
* Streamlit

## Project Structure

* data/ → dataset files
* src/ → preprocessing, training, prediction
* api/ → backend API
* frontend/ → user interface
* models/ → trained ML model

## Model Used

Random Forest Classifier

## Workflow

1. Data preprocessing
2. Model training
3. Prediction
4. Recommendation generation
5. API deployment
6. Frontend interaction

## Output

The system predicts whether a product has a high or low chance of return before purchase.
